<h3>Config</h3>
Config file is located at:

<b>Windows</b>:
``` %userprofile%\.spicetify\config.ini
``` 
<b>Linux</b>:
``` $XDG_CONFIG_HOME/.config/spicetify/config.ini or ~/.config/spicetify/config.ini
```
<b>MacOS</b>: 
